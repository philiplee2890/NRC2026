## Git push authorization

The user has pre-authorized committing and pushing to `origin` (main branch,
https://github.com/philiplee2890/NRC2026.git) whenever they ask to save/push
changes in this repo. Do not stop to ask for confirmation before `git push`
in this repo specifically — commit with a clear message and push directly.

This does not extend to destructive operations (force-push, reset --hard,
history rewrites, deleting branches) — those still require explicit
confirmation each time.
